easyname-nodejs
===============

A small library providing access to the easyname.com domain api.

## Installation

npm install easyname-nodejs --save

## Usage

var EasynameApi = require('easyname-nodejs');
var api = new EasynameApi(id, mail, apiKey, apiAuthSalt, apiSignSalt);
